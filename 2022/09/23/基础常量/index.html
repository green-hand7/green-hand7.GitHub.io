2022-09-23 java 845 字
java

常量

概念：

在程序的执行过程中，其值不会发生改变的量（数据）

常量的分类

字符串：被双引号所包裹的内容

整数常量 整数，例如：-10、0、88等

小数常量 小数，例如：-5.5、1.0、88.88等

字符常量 用单引号括起来的一个字符，例如：’a’、’5’、’B’、’中’等

布尔常量 布尔值，表示真假，只有两个值true和false

空常量 一个特殊的值，空值，值为null 除空常量外，其他常量均可使用输出语句直接输出。

注意：null空常量不能输出打印

public class hall{
	public static void main(String[] args){
		System.out.println("HelloWorld"); //字符串
		System.out.println(123);//整数
		System.out.println(12.3);//小数
		System.out.println('a');//字符
		System.out.println(true);//布尔值
		System.out.println(5>10);
	}
}
变量介绍

概念：变量就是内存中调度存储空间，空间中存储着经常发生改变的变量（数据）

变量定义的格式：

数据类型 变量名=数据值

数据类型：为空间中存储的数据加入类型限制。如 整数？小数？

变量名：自己要为空间起的名字

数据值：空间中要有存储数值

数据类型

java语言是强类型语言，对于每一种数据都给出了明确的数据类型，不同的数据类型也分配了不同的内存空间，所有他们表示的数据大小也是不一样的。

数据类型内存占用和取值范围

不同的数据类型也分配了不同的内存空间，所以他们表示的数据大小也是不一样的。

数据类型 关键字 内存占用（字节）
整数 byte short int long 1 2 4 8
浮点数 float double 4 8
字符 char 2
布尔 boolean 1

整数选int 小数选double

类（class） 接口（interface） 数组（[]）

变量的定义和使用

变量的定义

public class bl {
    public static void main(String[] args)  {
	    //格式；数据类型 变量名=数据值;
		//整数类型变量;
		int a=10;
		System.out.println(a);
		//小数
		double b= 1.1;
		System.out.println(b);
		//字符串
		char c = 'a';
		System.out.println(c);
		//布尔
		boolean d =1>2;
		System.out.println(d);	
	
	} 
   
}

变量的使用

根据变量的名字来进行使用

int a=10;
System.out.println(a);
a =20
System.out.println(a);

总结：变量就是内存中的空间，内部储存着不断发生变化的数据

变量的注意事项

  • 变量名不允许重复定义

  • 一天语句可以定义多个变量，但需要使用逗号进行分割

  • 变量在使用之前一定要进行赋值

  • 定义float和long变量注意事项

    定义float变量时：需要在数值后面加入f（大小写都可以）

    定义long变量时：需要在数值后面加入L（大小写都可以）
